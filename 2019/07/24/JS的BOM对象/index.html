JS的BOM和DOM对象

发表于 2019-07-24

BOM

Broswer Object Model:浏览器对象模型。bom对象属于前端开发中很实用但并不特别常用的对象，若不做笔记容易遗忘生疏。包括了以下重要对象：

·navigator

用以获取客户机（浏览器）信息，如浏览器名称、类型、版本等。例:
navigator.appName;//浏览器名称

·screen

屏幕信息，如height,weight等。例：
screen.weight;

·location

请求的url相关信息。
location.href;//url地址

·history

请求的url历史记录，用来实现页面的“前进”、“后退“等

·window（重要）

窗口对象，JS的顶层对象，上述bom对象都是window对象的成员,另外alert,confirm等也都是window的方法，平时的alert(“aaa”)其实是window.alert(“aaa”)的简写。几个重要的方法：
window.open();//打开新窗口
window.opener();//得到创建该窗口的窗口，即父窗口对象
setInterval(“alert(‘aaa’);”,”3000”);//定时器操作，每3000ms执行一次alert
setTimeout(“alert(‘aaa’);”,”3000”);//定时器操作，3000ms之后执行一次alert
clearInterval(vid);//清除id为vid的interval定时器，vid是setInterval的返回值
clearTimeout(vid);同上

DOM

document object model:文档对象模型。dom主要有几个地方需要注意下：
·当你用js写document.xxxx.xxx时，document指的是整个html/xml文件。
·获取子节点时，document.getElementById(“aa”).childNodes会因为浏览器不同造成解析结果的不同，例如有些浏览器会将空格回车也当做子节点。所以getElementsByTagName()是比较好的做法。
