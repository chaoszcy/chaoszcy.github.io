写给非科班非名校的我们

发表于 2019-07-23 更新于 2019-07-28

WARNING:这篇文章是写给吾等双非院校没学历没基础的转业朋友们看的，大佬们请及时退出以免辣到眼睛😂。

        先说个结论吧：如果你觉得做程序员就是BAT保底，英语专八打底，学校985起步，那只能说明你知乎看多了。
        在知乎，学Java要看<<Java编程思想>>，学算法要看<<算法导论>>，学大数据你得先花一个月时间学Linux再花两个月时间看Scala，学Spring请直接看源码。你问他xxx方向好不好找工作，得到的答案永远是“只要你够优秀岗位随便挑”，再问下去就是“人在美国，刚下飞机，年薪百万，刚接上的头，利益相关，匿了”。总而言之给我的感觉就是：刷知乎超过半小时就会开始怀疑自己是不是弱智。
        个人觉得，要学编程，学习能力先不去说它，但首先你得有信息的筛选和鉴别能力。我不是想否定或者批判知乎，可以说我的计算机学习路线全是知乎上看来的，没有知乎我甚至踏不进这个行业的门。但是不得不说，前几年知乎留下的那些高亮回复，80%并不是写给“一般人”看的，他们是写给“大佬们”看的。现在的风气有所改观，但你2019年搜”如何零基础自学Hadoop”看到的肯定还是2017左右的回答，就会得到我上面写的“一个月Linux,两个月Scala，一个月MySQL，一个月NoSQL。。。补一年基础我再教你怎么搭环境”类似这样的答案。所以在你疯狂寻找自学方案时，首先请保证找到的学习路线适合自己。

        为了证明自己不是瞎说我会来摆摆证据。15年从运营商辞职后，我以非全日制研究生的身份加入到了全日制研究生的班级中。当时的情况是这样的：在职研究生是我14年被朋友怂恿一起考的，形式是周末两天上课，目的是混个硕士学位方便在老东家涨薪升职。但结果15年年初在运营商不想干了，那正好借着研究生的机会专心学习然后转业了。可是在职的课程质量真的是上过才知道多坑，无奈之下我找学校商量，以选修形式帮我安排进全日制班级。虽然当时的我确实很屑，说句不好听的我的资历就是毫无疑问全年级垫底的那位，但身边的同学可都是正儿八经985研究生。
        然后我们来看看我当年那届985研究生毕业季战果如何：华为保底，半数BAT？想多了。那年软工145个学生，微软笔试进二轮的不超过3个，SAP宣讲会只签了两三个，进BAT的粗略估计大约就10人，进华为的大约只有15%-20%，更多的都是去了华三、烽火、大唐电信这些企业。倒是我这个零基础只有50%纯度的研究生有幸通过了华为的实习招聘。虽然最终因毕业后没有毕业证的问题（当时的在职研究生是单证形式）没能转正，但至少当时面试官认可了我的学习能力，这就证明了我们这些双非本科甚至零基础的转业生，其实并不比985学生差多少。

        （这里应当有段结束呈词，但我语言水平实在太屑了写不出来，等有空找老婆代笔😂）
